# 📊 Sales Performance Dashboard

## Project Overview
This Power BI dashboard analyzes sales performance using the Superstore dataset. It provides insights into sales, profit, product performance, regional distribution, and customer segments through interactive visualizations.

## Features
- KPI Cards (Total Sales, Total Orders, Profit Margin, Total Profit)
- Top 10 Products Analysis
- Sales by Category
- Monthly Sales Trend
- Regional Sales Map
- Profit by Sub-Category
- Sales by Segment
- Interactive Slicers (Region, Category, Segment, Date)

## Tools Used
- Power BI Desktop
- Power Query
- DAX
- Superstore Dataset

## Key Insights
- Identified top-performing products.
- Analyzed sales trends over time.
- Compared category-wise sales performance.
- Evaluated regional sales distribution.
- Measured profitability across sub-categories.
